🏏 Cricket Player Performance Analysis

CodeAlpha Data Analytics Internship – Task 3

📌 Overview

This project analyzes and visualizes cricket player performance using Python. It focuses on key metrics such as runs, wickets, and batting average to extract meaningful insights through data visualization.

🎯 Objectives
Visualize cricket performance data
Compare player contributions
Derive insights using charts and graphs
🛠️ Tools Used
Python
Pandas
Matplotlib
Seaborn
📊 Visualizations
Bar charts for runs and wickets
Pie chart for run contribution
Scatter plot for performance comparison
🔍 Key Insights
Visualization simplifies player performance comparison
Balanced players provide higher overall value
Data-driven insights support objective evaluation
🚀 How to Run
pip install pandas matplotlib seaborn
python cricket_analysis.py
